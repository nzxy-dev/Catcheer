How Install?:
-------------
Just Copy or Cut Or Move the folder "catcher" to your construct 2 plugin dir
Example:
-------
[C:\Program Files\Construct 2\exporters\html5\plugins]

Or 
[Your Main directory C2\exporters\html5\plugins]

//Ensure the folder plugin folder keep the name (catcher)

______________________________________________________________________________

How export to Catcher With Construct2 ?:
------------------------------------------
Just Download Catcher.zip and extract
go to construct 2 and export with html5 
puth the all the files (index.html,data.js,c2runtime.js,offline.js,etc etc..)
inside the folder "_game" In the extracted catcher folder
Or Select it directly from the dialog exported of construct 2

-(Recomended)
inside "_game" or your html extratcted game open index.html with notepad,VScodeStudio or 
Any plain text tool and search and delete this line;

[ alert("Exported games won't work until you upload them. (When running on the file:/// protocol, browsers block many features from working for security reasons.)"); ]

For remove the construct 2 default security alert
